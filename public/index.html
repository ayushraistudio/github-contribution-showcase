1. What is this?

This is a serverless API that generates dynamic SVG trophies based on your GitHub contribution stats (Active Days, Streaks, Total Contributions). It updates automatically every time you commit code.

2. Integration Guide

Simply copy the URL from above and replace YOUR_USERNAME with your actual GitHub username. Then, paste it into your profile README.md file like this:

[![My Stats](https://personal-trophy.vercel.app/api?username=your_username&theme=neon)](https://github.com/ayushraistudio)

3. Want to Contribute?

This project is Open Source! You can add new themes, fix bugs, or improve the logic. Fork the repository and submit a Pull Request.
