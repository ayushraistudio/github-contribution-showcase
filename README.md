## üÜö How is this different from other GitHub trophy tools?

Most GitHub trophy or stats tools rely on scraped data or focus mainly on visuals.

This project stands out because it:

* uses GitHub‚Äôs **official GraphQL API** for accurate and reliable data,
* avoids scraping and fake statistics,
* focuses on clean, readable SVG output,
* highlights meaningful metrics such as streaks and active days.

## üöÄ Why use this API instead of other tools?

* **Live Updates:** No manual updates required. Visitors always see your latest GitHub stats.
* **Fast Performance:** Powered by Vercel Serverless Functions for quick load times.
* **Click to Discover:** Clicking the trophy redirects users to this repository so they can set up their own.
